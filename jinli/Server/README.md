生产环境安装
1.如果是一台全新的机器，需要执行preinstall.sh，注意用安装机的证书文本替换{cert text}（证书一般生成在~/.ssh/id_rsa.pub）。该脚本会安装docker和docker-compose，并在目标机上添加安装机的证书，以便ssh证书登录。
2.配置redis和mongodb集群
3.配置pro_config.env文件
4.执行install_remote.sh
————————————————————————————————————————————————————————————————————————————————————————————————————————
执行dev_env.bat可以单独启动开发环境需要的容器
